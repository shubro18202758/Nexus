# NEXUS Knowledge Base

This directory is your personal knowledge base managed by the Notes skill.

## How it works

- Drop `.md` files here and Nanobot can search/read them
- Use "quick capture" to jot down thoughts to your daily journal
- Create structured notes with tags for easy retrieval
- The `journal/` subdirectory holds daily journals
